Hướng Dẫn Sử Dụng

1
Tải file lên

Kéo thả hoặc click chọn file. Hỗ trợ PDF, JPG, PNG, WEBP, HEIC...

2
Chờ xử lý tự động

AI sẽ tự phân loại tài liệu, xử lý ảnh (glare, nghiêng), OCR và trích xuất dữ liệu.

3
Xem kết quả

Bảng dữ liệu chi tiết với độ tin cậy từng dòng. Màu xanh = tốt, vàng = trung bình, đỏ = cần kiểm tra.

4
Export dữ liệu

Xuất ra Excel (có định dạng đẹp + sheet tóm tắt), CSV, hoặc JSON ngay lập tức.

Loại tài liệu được hỗ trợ

✓ PDF Mềm

PDF có text layer, xuất từ Word/Excel/phần mềm kế toán

✓ PDF Scan

PDF được tạo từ máy scan — tự động chuyển sang OCR

✓ Ảnh Scan

JPG/PNG từ máy scan phẳng, chất lượng tốt

✓ Ảnh Điện Thoại

Chụp bằng smartphone — AI tự chỉnh góc, tăng sáng

✓ Ảnh Lóa Sáng

Phản sáng, đèn chiếu — xử lý bằng CLAHE + Inpainting

✓ Ảnh Nghiêng

Trang chụp xiên/nghiêng — perspective warp tự động
